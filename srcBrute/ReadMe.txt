****** Compilation Instructions ******

To compile the assignment submission:
	1 - Navigate to the 'src' directory
	2 - Place (i.e. Copy/Paste) desired input test file in the 'src' directory; to test the program the input txt file 
		must be in the same directory as Main.java
	3 - Input the following command into terminal:
			javac Main.java
	4 - To test the program, input the following command into terminal:
			java Main <inputFile.txt> <outputFile.txt>
	Result: The solution will be output to a text file names <outputFile.txt>
